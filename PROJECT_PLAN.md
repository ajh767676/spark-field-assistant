# Spark Field Assistant

# Design Philosophy

This app is not a spreadsheet.

It is a guided field assistant that helps acquisition agents make faster and better buying decisions while standing inside a property.

Every feature should reduce mistakes, reduce taps, or save time.

# Guiding Principles

1. Mobile First
Everything is designed for one-handed use on a phone.

2. Offline First
The app must work even without cell service.

3. Fast First
The agent should never wait for the interface.

4. One Decision Per Screen
Avoid overwhelming the user with too much information.

5. Business First
Every feature should help Spark Homes buy better houses.

# Success Criteria

If someone has never used the app...

They should be able to complete an inspection in under 10 minutes with almost no training.

## Goal

Build the most intuitive mobile field inspection application for real estate acquisition teams.
